# PM Skills — Claude Code Plugin

A Claude Code plugin for Product Managers. Helps teams build the right things by challenging assumptions, enforcing strategic alignment, and producing clean PM artifacts.

## Install

**Step 1: Register this repo as a marketplace (one-time per person)**
```
/plugin marketplace add anishthim88/claude-pm-skills
```

**Step 2: Install the PM skill**
```
/plugin install pm@claude-pm-skills
```

## Update

When the skill is improved, pick up changes with:
```
/plugin update pm@claude-pm-skills
```

## Skills included

### `/pm` — PM Advisor

Invoke with any idea or problem statement. The skill will:
1. Challenge your assumptions
2. Run structured Q&A (one question at a time)
3. Ask which company objective the feature serves
4. Produce PRD, Executive Brief, Technical Brief, or Testing Plan

## Contributing

Improve the skill by editing `plugins/pm/skills/pm/SKILL.md` and opening a PR.
